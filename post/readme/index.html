阿尔贝·加缪读书笔记

《西西弗神话》、《鼠疫》、《异乡人》......

Posted by 揭晨璨 on Wednesday, January 9, 2019

加缪在20世纪顶住了历史潮流，独自继承着源远流长的醒世文学，他怀着顽强、严格、纯洁、肃穆、热情的人道主义，向当今时代的种种粗俗丑陋发起了胜负难卜的宣战。 ——萨特

《西西弗神话》

  • 活着，带着世界赋予我们的裂痕去生活，去用残损的手掌抚平彼此的创痕，固执地迎向幸福。因为没有一种命运是对人的惩罚，而只要竭尽全力就应该是幸福的。拥抱当下的光明，不寄希望于空渺的乌托邦，振奋昂扬，因为生存本身就是对荒诞最有力的反抗。

《鼠疫》

  • 要了解一个城市，比较方便的途径不外乎打听那里的人们怎么干活，怎么相爱，又怎么死去。
  • 人世间的罪恶几乎是由愚昧无知造成的，如果缺乏理解，好心能造成和恶意同样大的危害。

《置身于苦难与阳光之间》

  • 当人迷途时，充满活力也是浪费自己的时间。那些要背弃世界的人，随他们去吧。我不抱怨，因为我看着我诞生。
  • 因为在遗忘的深处，从我面前再现的那些时光中，还留有对纯粹激情的一种完美的回忆，对于悬浮于永恒之中的时刻的回忆。这是我身上唯一真实的东西，但我知道它总是太迟了。我喜欢看一个弯曲的动作，喜欢景色中一棵位置恰当的树。为了重建这全部的爱，我们只需这样一个细节就足够了：长久关闭的房间的味道，脚步的特殊声响。我就是如此，如果我喜欢表现我自己，最终我是我自己，那时因为只存在着使我们回归自身的爱。
  • 走向我的东西并不是对更加美好的日子的希望，而是对于一切、对我自己纯净而又原始的冷漠。但是，应该粉碎这过于柔软、过于容易的曲线。我需要我的清醒，是的。一切都是简单的，是人自己使事物变复杂了。
  • 为了改变自然的冷漠，我置身于苦难与阳光之间。苦难阻止我把阳光下和历史中的一切都想象为美好的，而阳光使我懂得历史并非一切。
  • 我永远是我自己的囚犯。
  • 人的一切不幸源于希望。

《局外人》

  • 在我所度过的整个那段荒诞生活期间，一种阴暗的气息从未未来前途的深处向我扑面而来，它穿越了尚未来到的岁月，所到之处，使人们曾经向我建议的所有一切彼此之间不再有高下优劣的差别了，未来的生活也并不比我以往的生活更真切实在。
  • 我的灵魂与我之间的距离如此遥远，而我的存在却如此真实。
  • 人永远都有一套哲理，来解释自己为何缺乏勇气。

《反抗者》

  • 我大喊我什么都不相信，一切都是荒诞的，但我不能怀疑我的呼喊，至少应该相信我的抗议。
  • 纯粹的自由乃是杀人的自由。

「真诚赞赏，手留余香」
